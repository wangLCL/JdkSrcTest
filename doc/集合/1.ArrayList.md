# ArrayList

ArrayList动态数组，可以加入删除元素等

## 字段

- 默认尺寸DEFAULT_CAPACITY=10
- 空数组EMPTY_ELEMENTDATA
- 存储数据的数组 elementData
- size存储的大小


## 构造函数

- public ArrayList(int initialCapacity)：
    - 用户给的大小小于0 ，抛出异常
    - 大于0，传入参数作为数组的大小
- public ArrayList() 
    - 将空数组给存储数组，此时并不具存储能力
- public ArrayList(Collection<? extends E> c)
    - 使用集合赋值
- 最大容量MAX_ARRAY_SIZE Integer.MAX_VALUE-8;
